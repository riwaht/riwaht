Front-End Developer and Designer
--------------------------------

I've been learning to code for about three years, starting with pure Java to branching to several fields, mainly game development and web development. I have also taken a unique path to learn design and animation, in order to bridge the gap between engineering and design. Thus, I've gotten into the game development aspect using Unity, C# and 3D designing using Blender. I have also used the above skills to work on VR projects. Meanwhile, I'm building up my front-end skills after learning HTML, CSS and JavaScript. Also, on my path to learn design, I've practiced on important Adobe software that allowed me to focus on the foundations of design in my applications. As I learned more about programming, I realized that most engineers overlook the design aspect of the code, instead focusing on its functionality. However, instead of leaving it to designers, I believe the work would be much better if left to the engineer himself/herself. I consider my work ethic to be integrating creativity and engineering into my projects, as it is necessary to include both for a near-perfect result!
